# יומן אישי 🌿

PWA יומן אישי בעברית עם תמיכה ב-RTL, צבעי פסטל רגועים ומעקב מצב רוח.

## תכונות
- כתיבת יומן יומית: תודה, כוח שהפגנת, דברים לזכור, מחשבות נוספות
- מעקב מצב רוח עם סלאיידר 1–9
- סיכום שבועי ומצג גרפי של מצב רוח
- טאב נפרד ל-AI review
- RTL ותמיכה בעברית
- פופ-אפ התראות יומיות
- תמונה רגועה בראש הדף
- שמירת נתונים באמצעות LocalStorage
- שדות טקסט מתרחבים בהתאם לתוכן
